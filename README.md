# MyFirstApplication - Android Beginner Submission

Aplikasi Android sederhana sebagai submission akhir kelas **Belajar Membuat Aplikasi Android untuk Pemula**.

## Fitur Utama
* **Main Page:** Menampilkan daftar Top 10 Bahasa Pemrograman menggunakan `RecyclerView`.
* **Detail Page:** Menampilkan detail informasi saat item diklik (Intent Parcelable).
* **Profile Page:** Halaman "About Me" sederhana.
